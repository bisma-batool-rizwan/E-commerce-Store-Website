# 🛍️ Nova E-Commerce Store

A modern and responsive e-commerce website designed to deliver a clean and engaging online shopping experience. Nova features a stylish landing page, featured products, category browsing, promotional banners, and a mobile-friendly layout for seamless browsing across all devices.

## 🌐 Live Demo

https://nova-e-commerce-store.netlify.app/

## ✨ Features

- Modern and responsive UI
- Hero banner with promotional offers
- Featured products section
- Product categories
- Product cards with pricing and ratings
- Mobile-friendly design
- Clean and intuitive user experience

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Responsive Web Design
- Netlify (Deployment)
